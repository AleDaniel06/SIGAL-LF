# SIGAL-LF — Sistema de Gestión de Almacén e Inventario

### 🏬 Proyecto: "La Fábrica" — Sucursal Huancayo

Sistema web diseñado para la gestión, control matricial (tallas/colores) y optimización del inventario de la tienda de ropa "La Fábrica".
